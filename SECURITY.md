# Security Policy

## Supported Versions

We currently support the latest release on the `main` branch. Older versions may receive security fixes on a best-effort basis.

## Reporting a Vulnerability

If you discover a security vulnerability in Recollect, please report it privately.

**Preferred**: Use GitHub's private vulnerability reporting at:

https://github.com/cobusgreyling/recollect/security/advisories/new

(Or open a security advisory from the repo "Security" tab.)

Alternatively, email the maintainer (see GitHub profile for contact if private reporting is unavailable).

Please include:

- Description of the vulnerability and impact
- Steps to reproduce (or a minimal PoC if possible)
- Suggested fix or mitigation, if known
- Your contact information for follow-up

We aim to acknowledge reports within 72 hours and will work with you on coordinated disclosure.

Thank you for helping keep the project (and its users) safe.
